Overview

In-memory cache implementation with support for lazy loading values. Provides a simple key-value store with cache miss handling.

def get(key)
  return @records[key] if self.include?(key)
  miss = nil
  miss = yield(key) if block_given?
  # don't cache nils
  return miss if miss.nil?
  @records[key] = miss
end
